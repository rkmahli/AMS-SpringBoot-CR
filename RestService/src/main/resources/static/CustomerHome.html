Welcome Customer

Agent Management System gives you the option to review your appointments or to schedule a new one.

View Appointments Schedule Appointments

View Appointments:
Upon clicking on the view appointment button, This module allows you to view all the appointments you have scheduled. It give you details like the time and date of appointment, agent name, etc. Furthermore, the appointments are divided into upcoming and past appoinments for convenience.
Schedule Appointment:
This module of Agent Management System allows you to schedule an appointment. It requires you to provide details like the date and time slot you prefer to have an appointment. It gives you a success message if the appointment request is successful.

About Us

This independent application, Agent Management System (AMS) website is used for insurance agent maintenance, license management, scheduling customer appointments for Insurance policies and agent commission calculation for an insurance consultant.

AMS is a system used for maintaining the details of the Insurance agent like adding the agents, managing the license that has been provided, to schedule customer appointments to know about the Insurance policies, to calculate the commission for each and every insurance agent. The following are the few important modules in the system
a) Agent Addition
b) License Management
c) Customer Appointments
d) Customer Policy Registration
e) Agent commission.
